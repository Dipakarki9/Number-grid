Diffrence between HTML AND CSS

S.N HTML CSS
1 HTML stands for hypertext markup language CSS stands for Cascading Style Sheet
2 HTML is used to create structure of a webpage CSS is used to design and style a webpage
3 HTML defines elements like text,images,links,tables CSS control colors,font,size,spacing,layout
4 HTML uses tags CSS uses selectors and properties.
5 HTML mixes content and structure. CSS separates design from content.
6 HTML files are with .html extension. CSS files are saved with .css extension.
7 HTML is simpler and static CSS is more flexible and dynamic.
8 HTML does not support animations. CSS supports animations and transitions.
9 HTML alone gives a plan look. CSS makes webpages attractive.
